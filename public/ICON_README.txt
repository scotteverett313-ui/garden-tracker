# Icon

You need a 512x512 PNG icon at `public/icon.png` for the iPhone home screen icon.

Quick options:
1. Use any square image you like and name it icon.png
2. Create a simple one at https://favicon.io/favicon-generator/
   - Text: 🌿 or "PT"
   - Background: #2d6a3f (dark green)
   - Download and rename to icon.png

Place it at: garden-tracker-app/public/icon.png
